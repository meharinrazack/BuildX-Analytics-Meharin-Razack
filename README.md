# BuildX-Analytics-Meharin-Razack
# Restaurant Analytics Project

## Dataset Selected

Restaurant Dataset

## Why I Selected It

I selected this dataset because it contains information about restaurants such as ratings, costs, cuisine types, online ordering, and table booking. It allows meaningful business analysis and dashboard creation.

## Three Business Questions Answered

1. Which rating category has the highest number of restaurants?
2. How many restaurants offer online ordering and table booking?
3. Which areas have the highest number of restaurants?

## Tools Used

- Python
- SQL
- Power BI
- GitHub

## One Key Insight

Most restaurants belong to the Average rating category, while online ordering is available in a significant number of restaurants.
